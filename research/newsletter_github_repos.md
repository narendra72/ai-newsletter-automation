# AI Newsletter Research Results – GitHub Repos

**Research Focus:** Agentic AI tools, frameworks, and agent skill systems (matches the newsletter's best-performing "GitHub Repos" section from Issue #2).

## 💻 Top GitHub Repos of the Week

1. **langchain-ai/openwiki** – A CLI tool that automatically writes and maintains AI-agent-readable documentation for a codebase, making it easier for coding agents to understand a project's structure.
   * ⭐️ Stars (since recent launch): ~2,700
   * 🔗 [https://github.com/langchain-ai/openwiki](https://github.com/langchain-ai/openwiki)

2. **Kulaxyz/self-learning-skills** – A self-improving "skill" for AI coding agents (Claude Code, Cursor, AGENTS.md-based tools) that captures a hard-won successful solution path from one session and reuses it in future ones — a direct, practical example of the "agentic skills" pattern covered in this week's featured research.
   * ⭐️ Stars (since recent launch): ~825
   * 🔗 [https://github.com/Kulaxyz/self-learning-skills](https://github.com/Kulaxyz/self-learning-skills)

3. **QwenLM/Qwen-AgentWorld** – Alibaba's Qwen team released "language world models" for general-purpose agents, aimed at giving agents a learned model of their environment to plan and act more reliably.
   * ⭐️ Stars (since recent launch): ~760
   * 🔗 [https://github.com/QwenLM/Qwen-AgentWorld](https://github.com/QwenLM/Qwen-AgentWorld)

4. **benchflow-ai/awesome-evals** – A curated, no-fluff library of papers, tools, and benchmarks for building and evaluating AI agents — a handy reference for teams trying to measure whether their agents actually work.
   * ⭐️ Stars (since recent launch): ~660
   * 🔗 [https://github.com/benchflow-ai/awesome-evals](https://github.com/benchflow-ai/awesome-evals)

5. **HKUDS/AgentSpace** – A "Human + Agents, One Team, One Workspace" project exploring shared collaborative workspaces where humans and multiple AI agents work side by side on the same tasks.
   * ⭐️ Stars (since recent launch): ~610
   * 🔗 [https://github.com/HKUDS/AgentSpace](https://github.com/HKUDS/AgentSpace)

*Note: star counts reflect approximate totals accumulated since each repo's recent creation date (used here as a proxy for "trending this week" activity via the GitHub Search API).*
